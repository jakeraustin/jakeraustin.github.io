Pac Man Scroller

Adrduino Project

Within this video my partner and I made sure we should two working functions of the Pac Man Scroller game. At the start of the game you can hear the famous Pac Man song. After the game starts you can see Pac Man himelf and a two sets of numbers on the left side. The number on the top indicates if the character is moving by going from 0 to 1. The number on the bottom idicates the number of times Pac Man has gone across the screen. All of these things help create a complete game.

Written Response

For this project my partner and I wanted to make a simple but fun game on the arduino circuit board. We created a scroller that allowed the user to press a button and have pac man scroll across the screen. While pac man goes across the screen a counter on the left of the screen indicates how many times pac man has gone across the screen and indicates when the button is being pressed. During the course of this project we ran into two issues, one with the button and an unknown trail seen after Pac Man moved. We fixed these by fixing code issues with Pac Man himself and using the in class example of how to use the button to ensure our own button worked.
